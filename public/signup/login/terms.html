Terms & Conditions

Welcome to the NIC DAK Logbook — a bilingual (English/Hindi) web-based system for managing government postal correspondence (Dispatch & Acquired records). Developed and hosted by the National Informatics Centre (NIC), Meghalaya State Center, Shillong.

By registering and using this application, you agree to the following terms and conditions.

1. Registration & Account Creation

Users must register by providing:

  • First Name & Last Name: Must be valid and accurate
  • Phone Number: Must be exactly 10 digits, containing only numerical values (0-9)
  • CAPTCHA Verification: Required to prevent automated/bot registrations
  • Terms Agreement: You must agree to these Terms & Conditions

Important: Duplicate accounts are strictly prohibited. The system performs case-insensitive name matching and checks for duplicate phone numbers. Each user may have only ONE active account. Attempting to create multiple accounts may result in account suspension.

Your account is personal and non-transferable. Do not share your login credentials with anyone else.

2. Login & Data Access

To access the application:

  • Login using your registered phone number
  • Your data is loaded from a secure PostgreSQL database specific to your account
  • Session tokens expire after 10 hours for security

Security: Your phone number is encrypted using industry-standard Argon2 hashing. Your credentials are never stored in plain text.

3. Application Features & Functionality

Dispatch Module (Outgoing Mail)

Track outgoing letters/documents sent to recipients:

  • Record: Recipient name, designation, address, subject, date, delivery mode
  • Bilingual support: All fields in English and Hindi (auto-translated)
  • Track copy recipients
  • Set priority (Immediate / Priority)
  • Select delivery mode (Hand, E-mail, E-file, Speed Post, Registered Post)
  • Save as Draft or Submit for final record

Acquired/Receipt Module (Incoming Mail)

Log incoming letters/documents received from other offices:

  • Record: Sender name, designation, subject, date, receipt method
  • Bilingual support: All fields in English and Hindi (auto-translated)
  • Track copy recipients
  • Set priority and receipt method
  • Save as Draft or Submit for final record

Search & Filtering

  • Multi-dimensional search: Zone, Priority, Delivery Mode, Date Range, Status
  • Full-text search: By subject, sender, recipient names
  • Column sorting: Sort by any field (ascending/descending)
  • Quick filters: Today, This Week, This Month, This Year
  • View up to 500 rows per page

Reports & Export

  • Single-entry PDF export: Print individual letters/documents
  • Batch reports: Monthly, Quarterly, Annual, Custom Date Range
  • Statistics: Volume trends, priority distribution, zone-wise breakdown
  • Audit-ready compliance reports for internal/external stakeholders

Bilingual Support

  • All major fields captured in both English and Hindi
  • Automatic English → Hindi translation via Python Deep Translator API
  • Graceful fallback: If translation fails, original text is retained
  • Respects India's bilingual communication requirements

Smart Features

  • Auto-Increment Serial Numbers: Each user's entries are numbered automatically (001, 002, 003...)
  • Pincode Lookup: Enter a pincode → auto-fill State, City, District
  • Draft/Pending Workflow: Save entries as Draft, finalize later
  • Timestamp Tracking: All entries record creation/modification time
  • Audit Trail: System logs all actions for compliance

4. Data Security & Compliance

Your data is protected by multiple layers of security:

  • Authentication: JWT-based session management with 10-hour token expiry
  • Password Protection: Argon2 hashing (industry-standard, computationally intensive)
  • Data Isolation: Each user's data is strictly isolated via user_id (per-user serial numbering)
  • Network Security: All external API calls (translation, pincode) are whitelisted and validated
  • HTTPS/TLS: All communications are encrypted
  • Database: PostgreSQL with secure connection pooling

Important: This application operates in a secure, restricted network environment (DMZ) where only critical APIs are accessible. No unauthorized external traffic is permitted.

6. Data Privacy & Storage

Your personal information includes:

  • First Name & Last Name
  • Phone Number (10 digits only)
  • Dispatch & Acquired records you create

Storage: All data is stored securely in a PostgreSQL database managed by NIC Meghalaya. Data is not shared with external parties without your consent, except as required by law or for operational purposes.

Retention: Your data is retained as long as your account is active. Upon account deletion (if applicable), data is retained per NIC's archival policies.

Compliance: This application complies with India's data protection guidelines and government information security standards.

7. Translation & External APIs

The application uses the following external services:

  • Python Deep Translator (HuggingFace): For English → Hindi translation
  • India Post Pincode API: For address lookup from pincodes

Note: These are the only external APIs your data is transmitted to. All other network traffic is blocked at the firewall level for security. Data sent to these APIs is limited to the minimum required (text for translation, pincode for address lookup).
